# NativeSharp
A .NET Framework library for interacting with Native C++ programs


# Features
```
🔄 - Native class mapping
	🔄 - Native class wrapping in C#
	❌ - Class fields (with modification)
		❌ - Pointer fields
		❌ - Primitive fields
			❌ - byte
			❌ - char
			❌ - short
			❌ - int
			❌ - float
			❌ - long
			❌ - double
	❌ - Class vtable functions
	❌ - Class functions (__thiscall) (Non vtable)
❌ - Function calling
❌ - Function hooking
```
